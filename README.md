# Datapush

This service made for the purpose of aggregation data from two other services (Contracts and Payments).
Its has several endpoints for each service:
  # Contracts:
    * contracts/
    * contracts/id
    * contracts/filter
    * contracts/payments
  # Payments:
    * payments/
    * payments/id
    * payments/filter
    * payments/contracts
  # General request:
    * contracts-payments
